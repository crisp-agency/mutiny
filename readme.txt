== Installation ==
1. Download a zip file of the latest version of the theme at
   https://github.com/crisp-agency/mutiny/releases/latest
2. In your WordPress admin panel, go to Appearance -> Themes and click the
   'Add New' button.
3. Click on the 'Upload Theme' button and upload the zip file.
4. Click on the 'Activate' link to use your new theme right away.
5. Navigate to Appearance > Customize in your admin panel and customize to
   taste.
